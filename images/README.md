# 图片资源

此目录用于存放项目截图和展示图片。

## 目录结构

- `screenshots/` - 项目界面截图
- `logo.png` - 项目 Logo（可选）
- `banner.png` - 横幅图片（可选）

## 上传图片

1. 将截图文件放入此目录
2. 提交并推送到 GitHub
3. 在 README.md 中使用相对路径引用

## 图片命名建议

- `login.png` - 登录页面
- `dashboard.png` - 仪表盘
- `user-management.png` - 用户管理
- `cardcode-management.png` - 卡密管理
- `invite-center.png` - 推广中心
- `admin-panel.png` - 管理员面板
